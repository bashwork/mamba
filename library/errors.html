Mamba Exceptions¶

The following are the exceptions used throughout the mamba queue library.

class mamba.errors.MambaException¶
Base exception for all exceptions in the mamba message queue library.
class mamba.errors.TransactionLogException¶
Exception that is thrown when there is undefined behaviour with the persisent queue transaction log.
class mamba.errors.QueueCollectionException¶
Exception that is thrown when there is undefined behaviour with the queue collection
