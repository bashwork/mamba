Mamaba Client¶

As mamba, like starling, speaks the “memcache” protocol, any memcache client should be sufficient for interacting with the server.

However, we also define a wrapper around the standard memcache wrappers (cmemcache if it is available and fall-back to the pure python version) to simplify client code.

class mamba.client.Client(servers, debug=0, pickleProtocol=0, pickler=<built-in function Pickler>, unpickler=<built-in function Unpickler>, pload=None, pid=None, server_max_key_length=250, server_max_value_length=1048576)¶

A simple wrapper around a memcache client to simplify the client code needed to be written.
