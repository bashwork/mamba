API Documentation¶

Implementation of a Twisted Mamba Server¶

Mamba is driven from the twisted asynchronous event/networking library. This allows mamba to be run using minimal resources while at the same time solving the c10k problem.

Example run:

from mamba.server import StartServer
StartServer() # this will not return
class mamba.server.MambaProtocol¶
