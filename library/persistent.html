persistent — Mamba Persistent Queue¶

Module author: Galen Collins <bashwork@gmail.com>

Section author: Galen Collins <bashwork@gmail.com>

API Documentation¶

class mamba.persistent.PersistentQueue(persistence_path, queue_name)¶

PersistentQueue is a subclass of Python synchronized class. It adds a transactional log to the in-memory Queue, which enables quickly rebuilding the Queue in the event of a sever outage.

close()¶

Finish all writes to this queue’s transaction log file and close it.

Returns:void
get(log=True)¶

Retrieve the next element off the queue

Parameter:log – Set to True to log to the transaction log, False otherwise
Returns:The next item off of the queue
purge()¶

Purge the entire transaction log for this queue

Returns:void
put(value, log=True)¶

Pushes value to the queue. By default, put will write to the transactional log. Set log to False to override this behaviour.

Parameters:
  • value – The value to queue up
  • log – Set to True to log to the transaction log, False otherwise
Returns:

void
