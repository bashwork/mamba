Mamba Protocol Handler¶

class mamba.handler.Handler(database, statistics)¶

Class to abstact away all the handling for the mamba server protocol. Since we are using the twisted line reader protocol, we shouldn’t have to do any network work here, simply the logic for the protocol. To abstract it even more, the response command is passed in as a continuation.
